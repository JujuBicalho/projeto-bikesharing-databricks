##Pipeline de Engenharia de Dados e Análise de Dados##
(Bike Sharing Analytics)

##Sobre o Projeto: Construção de um Lakehouse completo no Databricks utilizando a arquitetura Medalhão (Bronze/Silver/Gold) para analisar dados de aluguel de bicicletas. O objetivo foi transformar dados brutos em KPIs de negócio para tomada de decisão.

=>Tecnologias Usadas:

Databricks (Processamento)

PySpark (Limpeza e Tratamento na camada Silver)

SparkSQL & Window Functions (Análises complexas na camada Gold)

Delta Lake (Versionamento e Time Travel)

=>Destaques Técnicos:

- Otimização de performance com ZORDER.

- Testes de Qualidade de Dados (Data Quality) automatizados.

- Governança de dados com documentação via catálogo.
